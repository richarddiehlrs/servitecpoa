# Publicar com GitHub Desktop — repositório `servitecpoa2`

## Passo a passo

1. Abra o **GitHub Desktop**
2. Menu **File** → **Add local repository**
3. Clique em **Choose...** e selecione:
   ```
   C:\Users\Davi\Documents\GitHub\servitecpoa2
   ```
4. Se pedir para criar repositório, confirme **create a repository**
5. Na lista de arquivos alterados, preencha o resumo do commit, por exemplo:
   ```
   Site ServitecPoa — consertos em eletrodomésticos
   ```
6. Clique em **Commit to main**
7. Clique em **Publish repository**
8. Nome do repositório: **servitecpoa2**
9. Deixe desmarcado "Keep this code private" se quiser site público (recomendado para Vercel)
10. Clique em **Publish repository**

Pronto. O código estará em: `https://github.com/SEU_USUARIO/servitecpoa2`

## Deploy na Vercel

1. Acesse https://vercel.com → **Add New Project**
2. Importe o repositório **servitecpoa2**
3. Deploy automático (Next.js)

## Atualizar depois

Sempre que mudar o site em `C:\site servitecpoa`, copie de novo para `Documents\GitHub\servitecpoa2` ou trabalhe direto nessa pasta — depois **Commit** e **Push origin** no GitHub Desktop.
