# GitHub — repositório servitecpoa2

Repositório: **https://github.com/richarddiehlrs/servitecpoa2**

## Atualizar o código no GitHub

No GitHub Desktop, com o repositório **servitecpoa2** aberto:

1. Revise as alterações
2. Resumo do commit → **Commit to main**
3. **Push origin**

## Clonar em outro computador

```bash
git clone https://github.com/richarddiehlrs/servitecpoa2.git
cd servitecpoa2
npm install
npm run dev
```
